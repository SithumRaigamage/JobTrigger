## 7. Accessibility

### 7.1 VoiceOver Support

| Element | Label | Hint |
|---------|-------|------|
| Job row | "{Job name}, {status}" | "Double tap to view details" |
| Trigger button | "Build now" | "Double tap to trigger build" |
| Status badge | "{status}" | - |
| Favorite star | "Add to favorites" | "Double tap to toggle" |

### 7.2 Dynamic Type

- All text scales with system settings
- Minimum font size: 12pt
- Maximum font size: 40pt
- Layout adapts to larger sizes

### 7.3 Color Accessibility

- All colors meet WCAG 2.1 AA contrast (4.5:1)
- Status communicated with icons, not just color
- Support for color blindness modes
